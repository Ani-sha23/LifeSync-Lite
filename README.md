# LifeSync Lite

LifeSync Lite is a lightweight and efficient Java-based productivity application designed to help users manage their daily life in a simple and organized way. It provides essential tools for task management, note-taking, and basic life tracking — all in one place.

## ✨ Features
- 📌 Task Management (Add, View, Delete tasks)
- 📝 Notes Management
- 💰 Basic Expense Tracking
- 📊 Simple and clean console interface
- ⚡ Lightweight and fast performance

## 🎯 Objective
The goal of this project is to build a simple yet effective life management system using core Java concepts such as classes, objects, file handling, and modular design.

## 🛠️ Tech Stack
- Java (Core Java)
- OOP Concepts
- File Handling

## ▶️ How to Run
1. Compile the program:
   javac src/Main.java

2. Run the program:
   java -cp src Main
